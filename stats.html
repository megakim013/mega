🔮 마법명중 및 디버프 적중확률 공식
1. INT 보정값 = (현재 INT - 45) × 0.5
2. INT 표 명중 = INT 레벨별 마법명중 보정값 (스탯 계산식 INT표 기준)
3. 최종 명중% = 기본 명중% + INT 보정값 + (INT 표 명중 + 아이템 마법명중 - 15)
4. 최종 적중% (MR 보정) = 최종 명중% - (대상 MR 수치 × 0.8)
※ 서버 기본 밸런스 설정: 대상의 마법 방어력(MR) 1당 디버프 적중확률이 0.8% 감소합니다.
※ 아이템 마법명중 1당 최종 명중률이 1%씩 상승합니다.

💡 디버프 적중확률 상세 공식 대입 예시

[예시 1] INT 40, 대상 MR 60인 상대에게 턴언데드 등 디버프 시전 시
  • INT 보정값 = (40 - 45) × 0.5 = -2.5
  • INT 표 명중 = +8 (보정 공식 적용: 8 - 15 = -7)
  • 최종 명중% = 100 + (-2.5) + (-7) = 90.5%
  • 상대 MR 60에 따른 감소치 = 60 × 0.8 = 48%
  • ★ 실제 최종 적중확률 = 90.5% - 48% = 42.5%
[예시 2] INT 45, 대상 MR 60인 상대에게 턴언데드 등 디버프 시전 시
  • INT 보정값 = (45 - 45) × 0.5 = 0
  • INT 표 명중 = +10 (보정 공식 적용: 10 - 15 = -5)
  • 최종 명중% = 100 + 0 + (-5) = 95%
  • 상대 MR 60에 따른 감소치 = 60 × 0.8 = 48%
  • ★ 실제 최종 적중확률 = 95% - 48% = 47%
[예시 3] INT 50, 아이템 마법명중 +10 보유, 대상 MR 60 상대에게 디버프 시전 시
  • INT 보정값 = (50 - 45) × 0.5 = +2.5
  • INT 표 명중 + 아이템 마법명중 = 14 + 10 = 24 (보정 공식 적용: 24 - 15 = +9)
  • 최종 명중% = 100 + 2.5 + 9 = 111.5%
  • 상대 MR 60에 따른 감소치 = 60 × 0.8 = 48%
  • ★ 실제 최종 적중확률 = 111.5% - 48% = 63.5%

🔥 SP (스펠파워) 합산 및 마법데미지 적용 공식
1. Total SP = 아이템 SP + 레벨 SP + INT 보너스(magicBonus)
2. 마법 최종 데미지 = INT 마법데미지(INT표 기준) + Total SP
※ SP는 곱연산이 아닌 덧셈(Additive) 연산으로 최종 마법데미지에 직접 수치만큼 더해집니다.
